RELATÓRIO

O site tem como objetivo fornecer uma boa experiência de compra para o usuário, num ambiente o mais minimalista possível e de fácil uso, com isso, temos como publico alvo pessoas de todas as idades. Utilizamos para a confecção do site HTML5, CSS e JavaScript. Para que o usuário tenha a melhor experiência é necessário que ele tenha acesso a internet e, apesar de intuitivo, ter um mínimo de conhecimento sobre como acessar sites. Como ponto positivos temos que todas as imagens utilizadas no nosso site foram criadas a partir de uma inteligência artificial chamada MidJourney. Como ponto negativo temos que nosso site não está hospedado. Encontramos dificuldade para implementar a responsividade, visto que existem diversos dispositivos com tamanhos diferentes. As atualizações serão feitas semanalmente, acompanhando as tendências do mercado.
